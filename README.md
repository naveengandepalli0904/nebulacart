Prerequisites:-

Install Node.js, VS Code and Git
Create accounts on GitHub, MongoDB Atlas, Cloudinary, Railway and Vercel

Setup Instructions:-

Download and extract nebulacart.zip
Open the nebulacart folder in VS Code
Create .env file and write your MongoDB, JWT and Cloudinary details
Open terminal and run npm run install-all
Write your backend code in backend/ folder (server, models, routes)
Write your frontend code in frontend/src/ folder (pages, components, context)
Connect frontend to backend through utils/api.js
Run node backend/seeder.js to add sample data
Run npm run dev and test on http://localhost:3000
Push code to GitHub
Deploy backend on Railway and add environment variables
Deploy frontend on Vercel and add REACT_APP_API_URL
Run seeder from Railway console for live database
